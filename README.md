# 📝 To-Do List App

A simple web-based To-Do List application that allows users to add, edit, and delete tasks.

This project was built using:
- ✅ HTML
- ✅ CSS
- ✅ JavaScript

## 💡 Features

- ➕ Add new tasks
- 🖊️ Edit existing tasks
- ❌ Delete tasks
- 🎯 Clean and responsive design
